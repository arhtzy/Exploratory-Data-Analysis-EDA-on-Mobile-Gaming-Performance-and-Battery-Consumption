#  Mobile Gaming Performance Analysis

This project focuses on Exploratory Data Analysis (EDA) to analyze mobile gaming performance based on FPS, session duration, and battery consumption across Android and iOS devices.

## Project Objectives
The main goals of this project are:
- Compare gaming performance between Android and iOS
- Analyze the relationship between FPS and battery consumption
- Explore player session duration patterns
- Identify games with the highest battery usage
- Discover correlations between gaming performance features

## Dataset Features
The dataset contains several important features, including:
- FPS
- Session_Time_Minutes
- Battery_Start_%
- Battery_End_%
- Battery_Drop_%
- Device_Type
- Game_Name

## Visualizations Used
This project includes several data visualizations such as:
- Histogram
- Line Chart
- Scatter Plot
- Boxplot
- Correlation Heatmap
- Bar Chart

## Tools & Libraries
Technologies and libraries used in this project:
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Plotly

## Key Insights
Some insights discovered from the analysis:
- Higher FPS tends to increase battery consumption
- Gaming performance differs between Android and iOS devices
- Certain games consume significantly more battery power than others
